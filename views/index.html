Ganesh API Automation Hub 🚀

Practice APIs for QA testers – with Auth, File Upload, Search & More

🔐 Authentication

  • POST /auth/login - Login and receive token
  • GET /profile - Access secured user profile

👤 User Management

  • GET /users?page=1 - Paginated user list
  • GET /users/:id - Get user by ID
  • POST /users - Create new user
  • PUT /users/:id - Update user
  • PATCH /users/:id - Partially update user
  • DELETE /users/:id - Delete user
  • GET /users/search?name=ganesh - Search user by name

📤 File Upload

  • POST /upload - Upload a file

🧪 Testing Utilities

  • GET /delayed - Simulate delayed response
  • GET /fail - Simulate 500 error
